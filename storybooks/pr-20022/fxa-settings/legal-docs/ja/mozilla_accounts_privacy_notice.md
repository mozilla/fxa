# Mozilla アカウントのプライバシーに関する通知

発効日 2026 年 2 月 10 日
{: datetime="2026-02-10" }

## Mozilla は、プライバシーが健全なインターネットの基盤であると信じています

そのため Mozilla では、あなたがオンライン上で共有する情報および Mozilla と共有する情報をより広範囲に制御できるよう、当社の製品を設計しています。このプライバシーに関する通知では、Mozilla がどのデータをどのように、どんな目的で処理しているかをご説明します。その後、ご自身の情報の処理方法を選んで指定する方法についてご説明します。

## Mozilla のデータの処理方法

### Mozilla アカウントをあなたに提供するため

Mozilla アカウントを使うと、1 回のログインで複数のデバイスからMozilla のサービスにアクセスできます。これを可能にするために、Mozilla ではあなたのアカウントデータ、連絡先データ（メールアドレス）、および認証データを処理させていただきます。皆様は、表示名やプロファイル画像などの追加のコンテンツデータを提供することも選択できます。

Mozilla では、Mozilla アカウントおよびそれに連携されたサービスの中核的機能を提供し、新しいデバイスを接続し、設定を記憶・尊重するため、さらに同期や組み込み VPN といったデフォルトの機能を提供する目的で、技術データや設定データなどの[特定のデータ](#bookmark-how-we-use-data)も収集します。

同期を有効にすると、Mozilla はデバイス全体で同期される情報を暗号化形式で受け取ります。これには、Firefox のタブ、アドオン、パスワード、フォームの自動入力情報、ブックマーク、履歴、設定などが含まれる場合があります。この暗号化はエンドツーエンドです。つまり、データはブラウザーを離れる前に暗号化され、暗号鍵を持つ別の同期クライアントのみが解読可能です。Mozilla でこの情報を解読することはできません。

有料のサブスクリプションが必要なサービスを購読した場合、お支払いは [Mozilla サブスクリプションサービスのプライバシーに関する通知](https://www.mozilla.org/privacy/subscription-services/)に従って処理されます。Mozilla は、皆様のアカウントの記録（請求先住所とお支払い方法の下4桁を含む）およびアカウントのサブスクリプションの状態を受信します。Mozilla が皆様のお支払い情報すべてを保管することはありません。

*その他の詳細 (データの設定を変更する方法など): * [Mozilla アカウントの管理方法](https://support.mozilla.org/kb/firefox-accounts-managing-account-data)や[同期を有効にする方法](https://support.mozilla.org/kb/sync-firefox-desktop)、[同期の動作についての詳細](https://support.mozilla.org/kb/sync)、およびアカウントに対して Mozilla が収集する[データ](https://dictionary.telemetry.mozilla.org/?page%3D1%26search%3Dmozilla%2520accounts)の詳細について確認することができます。

### Mozilla アカウントおよび連携サービスを提供・改善するため

また、Mozilla は、Mozilla アカウントの操作性を維持し、機能やパフォーマンスを改善し、問題を特定・解決・診断するためにデータを処理する必要があります。このために、技術データ、位置データ、設定データのほか、対話やシステムのパフォーマンスデータを使用します。

Mozilla では、技術データや対話データを処理することで、皆様がどのように当社のサービスに関与しているかについて理解を深めています。たとえば、[Mozilla アカウントのウェブサイト](https://accounts.firefox.com/)や[ヘルプページ](https://support.mozilla.org/)、ダッシュボード、メニュー設定への訪問状況、Mozilla アカウントに関連して使用する製品やサービス（Firefox の使用状況、使用頻度や最後にデータを同期した日時を含む）、Mozilla とのやり取りに使用したデバイス、さらに Mozilla からのメールや SMS メッセージに対する反応などが含まれます。これにより、Mozilla は、ユーザーが Mozilla のサービスにどのようにアクセスし、どのように使用しているかを理解し、ビジネスと当社のサービスの機能の改善に役立てることができます。

連携サービスを利用した場合、そのユーザーのデータは関連サービスのプライバシーに関する通知に従って処理されます：

* [Firefox ブラウザー](https://www.mozilla.org/privacy/firefox/)
* [Mozilla VPN、Mozilla Monitor、Firefox Relay を対象とする Mozilla サブスクリプションサービス](https://www.mozilla.org/privacy/subscription-services/)
* [MDN Plus](https://www.mozilla.org/privacy/mdn-plus/)
* [Thunderbird](https://www.mozilla.org/privacy/thunderbird/)

Mozilla アカウントの連携サービスで予期しない問題（例：クラッシュ）が発生した場合、Mozilla はサードパーティのサービスを利用して、エラーメッセージやデバイス情報、クラッシュ発生時のアプリケーションの状態などの技術データや対話データを収集して、Mozilla のサービスにおける問題を診断し、修正するのに役立てます。

*その他の詳細 (データの設定を変更する方法など): * [Mozilla アカウントデータ収集設定](https://support.mozilla.org/kb/firefox-accounts-managing-account-data)はいつでも変更できます。連携サービスに関連したデータ設定については、該当する通知をご覧ください。

### セキュリティを提供し、向上させるため

Mozilla では、技術データや対話データを使用して、Mozilla アカウントへのアクセスを安全に保ち、暗号化データを転送して同期できるようにし、不審な活動を検知するのに役立てています。Mozilla では、ウェブアプリケーションや [DDoS](https://en.wikipedia.org/wiki/Denial-of-service_attack) 攻撃からの保護を高めるために、サードパーティのウェブアプリケーションファイアウォールも利用しています。Mozilla のサーバーに保管されるデータはすべて暗号化されています。

### データを仮名化・匿名化・集計するため

Mozilla は、Mozilla アカウントの使用状況を理解する、マーケティングを測定するなど、前述のアクティビティの一環として、仮名化、匿名化、または集計したデータを作成することがあります。場合によっては、調査に役立てるため、または前述の合法なビジネス目的の一環として、集計・匿名化したデータを共有または公開することがあります。

### 皆様と連絡を取るため（当社サービスのマーケティング時も含む）

Mozilla では、皆様に連絡を取るために、連絡先情報を利用します。Mozilla では、皆様のアカウントで不審な活動を検知した場合の警告、アカウント情報の変更確認、マーケティング関連の連絡、カスタマーサポートへの対応などの目的で連絡を取らせていただくことがあります。また、皆様との連絡管理のためにサードパーティのサービスプロバイダーを利用することがあります。Mozilla では、これらの連絡に関連する対話データも処理し、誰が受信したかをより正確に把握できるようにしています。

### 適用法に従い、有害、不正、または不法なアクティビティを特定・防止するため

Mozilla は、適用法およびデータ保護目的に沿うためにあなたの個人データを処理することを求められる場合があります。たとえば

* [法的要求](https://www.mozilla.org/about/policy/transparency/#dfn-legal-process)に応えるため、および[法的手続き](https://www.mozilla.org/about/policy/transparency/) (たとえば、政府当局からの召喚令状、取り調べ、または要求に応えるなど) に従うため。Mozilla に対し、特定のユーザーデータを政府に開示することを強制するには、有効な法的手続きが必要です。
* 個人データにアクセスする権利や個人データを削除する権利など、プライバシー権の行使を求めるリクエストに応えるため。
* 詐欺かも知れないアクティビティ、または、不法行為、サイバー攻撃、知的財産の侵害などの有害なアクティビティを特定・調査し、それに対処するするため (法的要求の提出または弁護を含む)。
* コンテンツのモデレーションおよび関連した信頼性・安全性アクティビティを実行するため。
* 監査や企業セキュリティ管理などの社内コンプライアンスおよびセキュリティアクティビティを実行するため。

## データの国際的な転送

Mozilla Corporation (とそのサービス) は、米国を本拠としています。Mozilla はグローバルな会社であるため、このプライバシーに関する通知で言及しているサービスを提供するために、収集した情報に各地の従業員がアクセスする場合があり、また、弊社パートナーと共有した結果、それらのデータが他の地域に転送されることもあります。データが国を越えて転送される場合、Mozilla は、必要に応じて、あなたの個人データを保護するために適切なメカニズム (欧州委員会その他の該当する機関によって承認された[標準契約条項](https://commission.europa.eu/law/law-topic/data-protection/international-dimension-data-protection/standard-contractual-clauses-scc_en#eu-standard-contractual-clauses)など) を設置することを保証します。国際転送の際に実施される保護措置の情報、または関連する標準契約条項のコピーをご希望の場合は、[dpo@mozilla.com](mailto:dpo@mozilla.com) までご連絡ください。

## データの保持期間

Mozilla は、あなたの個人データを、このプライバシーに関する通知で概説した目的を果たすのに必要な期間のみ、保持します。一般に、アカウントがアクティブである限り、アカウントデータは保持されます。Mozilla では、2 年以上サインインされていないアカウントを[定期的に削除](https://support.mozilla.org/kb/mozilla-account-deletion-inactivity)します（この措置を取る前に、事前に通知をお送りするよう努めます）。

Mozilla が収集する技術データ、設定および対話データについては、一般に、25 か月以上保持されることはありませんが、実際の保持期間は、データタイプや収集の目的によって異なります (たとえば、あなたがいつから Mozilla アカウント保持者なのかを把握するため、「初回開設」の日付は、あなたがアカウントを使用する期間にわたって保持されます)。具体的な保持期間は、データの機密性、収集のコンテキスト、あなたの設定にもより、さらに、詐欺の防止、規制への準拠、またはサービスの継続などのために法的または契約上 Mozilla に課される、あなたのデータを保持する義務または削除する義務によっても異なります。Mozilla は、保持期間が過ぎた時点で、法律によって保持が求められない限り、あなたのデータを安全に削除します。

また、Mozilla のサービスは、災害時の復旧をサポートするため、暗号化したバックアップストレージを使用して保護を強化します。これらの環境におけるデータは、使用不能にされ、事業継続の目的にのみ処理されます。Mozilla は、データを、匿名化または集計した形で長期間保持することもあります。

## あなたの権利と選択肢

### あなたの権利

次のことに対する権利があります:

1.Mozilla があなたに関するどのデータをどのような理由で処理し、誰と共有しているのかを知る権利 (この通知がそれです!)
2.Mozilla が持っているあなたに関するデータについて、コピーを要求する権利
3.データの移植性を要求する権利
4.Mozilla が持っているあなたに関するデータのうち、不正確または不完全であるものについて、訂正を要求する権利
5.Mozilla が持っているあなたに関する個人データを (特定の条件下において) 削除させる権利
6.場合によっては、Mozilla によるあなたの個人データの使用を制限する、または使用に異議を唱える権利
7.Mozilla による個人データの扱いに懸念がある場合に当該のデータ保護機関に苦情を提出する権利。Mozilla は、まず Mozilla に ([dpo@mozilla.com](mailto:dpo@mozilla.com) を介して) 連絡してくださることを望んでいますが、当該の [EU データ保護機関](https://www.edpb.europa.eu/about-edpb/about-edpb/members_en)に連絡するか、地域のデータ保護機関を検索 (して連絡) することもできます。

個人データに関して上記のいずれかの要求を行う場合は、Mozilla Corporation まで、[データ主体アクセス要求ポータル](https://privacyportal.onetrust.com/webform/1350748f-7139-405c-8188-22740b3b5587/4ba08202-2ede-4934-a89e-f0b0870f95f0)を通じてご連絡ください。プライバシーの権利を行使したがために Mozilla から差別を受けることは絶対にありません。個人データまたは Mozilla のプライバシー慣行について他にご不明な点がある場合は、[compliance@mozilla.com](mailto:compliance@mozilla.com) までお問い合わせください。

## データの定義

| データタイプ | 説明 | Mozilla アカウントユーザーの例 |
| ----- | ----- | ----- |
| アカウントデータ | 登録情報など、Mozilla アカウントの管理に関連したデータ。 | アカウントのユーザー名、表示名、Mozilla アカウント ID。  |
| 認証データ | Mozilla では、あなたのアカウントをセキュリティ保護するためのデータを受け取るとともに、アカウントのセキュリティを維持するための認証関連のデータを作成します。 | パスワードハッシュ、tokenData。 |
| 連絡先データ | アカウント登録の一環として提供されるメールアドレスに加え、任意で Mozilla に提供されるその他の連絡先情報。 | メールアドレス、二次メールアドレス、電話番号。 |
| 対話データ | あなたが Mozilla のサービスをどのように利用しているかに関するデータ。 | クリック数、イベントログ、使用状況データ。  |
| 言語設定 | Mozilla のサービスとの対話にあなたが使用していると推測される言語、または設定言語 (設定で選択されている場合) | 言語。  |
| 位置情報 | Mozilla は、IP アドレスから国、都市、または地域レベルであなたの位置を推測します。 | 国コード、都市。  |
| 設定 | プライバシー設定など、サービスの提供方法に関するあなたの設定。特に選択を行っていない場合は、デフォルト設定が使用されます。 | プライバシー設定、アカウント設定。  |
| システムパフォーマンスデータ | サービスがあなたのデバイスでどのように機能しているかを示すデータ。 | タイムスタンプ、送信の成功/失敗。  |
| 技術データ | あなたが Mozilla のサービスへのアクセスに使用しているハードウェアやその構成、および接続に関する情報。 | デバイスのオペレーティングシステム、ブラウザーおよびバージョン、タイムスタンプ、ロケール。 |
| 一意識別子 | サービスとの対話を管理するためにさまざまな時点で生成される一意の識別子。 | Accounts_id、session_id。  |
