Summary
test(functional): upgrade Playwright to 1.61.1 and stabilize Sync/OAuth/CMS tests
Description
Because:
- Playwright 1.44.1 is outdated; its bundled Firefox no longer matches
  current browser behavior, and CI browser images need matching versions.
- Firefox 147+ now sends the keys_optional capability, so Relay and
  SmartWindow desktop signup no longer render a password-confirmation
  field, causing those tests to time out.
- Several Sync/OAuth/CMS tests raced with teardown or lacked navigation
  guards after the /pair routing changes.

This commit:
- Upgrades @playwright/test to 1.61.1 and bumps CircleCI browser images.
- Centralizes Sync entry through a real /pair handshake via gotoSyncSession
  and updates the affected Sync/OAuth/CMS/passkey specs.
- Removes the stale confirm-password step from Relay/SmartWindow signup.
- Adds navigation guards (waitForURL to settings/pair) to fix teardown
  races and documents the WebChannel responder duplicate-listener caveat.
